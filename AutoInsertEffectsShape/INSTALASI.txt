================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Plugin tidak load
SOLUTION: 
  - Check file path benar
  - Check System Monitor untuk error
  - Verify files tidak corrupt

PROBLEM: Macros tidak muncul
SOLUTION:
  - Refresh Macro Pool (F5)
  - Re-load plugin
  - Check System Monitor

PROBLEM: Effect tidak bergerak
SOLUTION:
  - Verify fixtures support Pan/Tilt
  - Check DMX output
  - Check BPM rate (mungkin terlalu lambat)
  - Verify effect applied ke fixtures yang benar

PROBLEM: Circle tidak sempurna
SOLUTION:
  - Check fixture calibration
  - Adjust wing values
  - Verify Pan/Tilt range fixtures

PROBLEM: Symmetric effect tidak simetris
SOLUTION:
  - Check fixture numbering
  - Verify fixtures dalam group yang sama
  - Check phase distribution

================================================================================
TIPS & TRICKS
================================================================================

TIP 1: Kombinasi Effects
  - Layer multiple effects untuk gerakan kompleks
  - Combine Circle + Pan untuk variasi
  - Adjust BPM untuk sinkronisasi

TIP 2: Wing Values
  - Wing kecil (-2,2) = tight pattern
  - Wing besar (-4,4) = wide spread
  - Experiment dengan values berbeda

TIP 3: Phase Ranges
  - 0-90 = subtle movement
  - 0-180 = balanced movement
  - 0-360 = full rotation

TIP 4: BPM Adjustment
  - Slow (30-40) = smooth, elegant
  - Medium (50-70) = balanced
  - Fast (80-120) = energetic

TIP 5: Symmetric vs Non-Symmetric
  - Symmetric = balanced look
  - Non-Symmetric = dynamic look
  - Mix keduanya untuk variety
