================================================================================
TROUBLESHOOTING:
================================================================================

Q: Plugin tidak jalan?
A: - Check file ada di folder plugins yang benar
   - Check System Monitor untuk error
   - Pastikan ada fixtures yang sudah di-patch

Q: Effects tidak muncul?
A: - Buka Effect Pool window
   - Refresh dengan F5
   - Check System Monitor untuk error

Q: Masih ada error "OBJECT DOES NOT EXIST"?
A: - Pastikan menggunakan version 2.0 (yang sudah di-fix)
   - Delete file lama dan copy file baru
   - Restart GrandMA2 jika perlu

Q: Beberapa effects gagal dibuat?
A: - Normal, beberapa form mungkin tidak compatible
   - Check System Monitor untuk detail
   - Effects yang berhasil tetap tersimpan

Q: Mau ubah effects yang dibuat?
A: Edit file auto_insert_effects.lua bagian "effects_config"
